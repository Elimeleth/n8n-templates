# Configurar Elevenlabs en community-nodes
1. Ir a [Nodos de comunidad](http://localhost:5679/settings/community-nodes)
2. Click en `INSTALL`
3. Pegar en el campo: `"npm Package Name"`: `@elevenlabs/n8n-nodes-elevenlabs`
4. Aceptar el riesgo y instalar
5. Esperar unos segundos

# Obtener apikey elevenlabs
1. Ir a [Elevenlabs](https://elevenlabs.io/app/developers/api-keys)
2. Crear Apikey
3. Configurar nodo `Elevenlabs` en n8n
4. Obtener el id de una voz [Voice ID](https://elevenlabs.io/app/agents/voice-library)
